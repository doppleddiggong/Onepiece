# Onepiece Agent 공통 가이드 {#onepiece_agent_common}

이 문서는 Onepiece 저장소에서 동작하는 모든 에이전트(Gemini, Claude, 범용 에이전트)가 공유해야 하는 프로젝트 맥락 정보를 한곳에서
참조하기 위한 허브입니다. 에이전트별 가이드는 본 문서를 링크하여 중복 서술을 피하고, 최신 프로젝트 규칙과 일관성을 유지합니다.

## 필수 참고 문서

| 문서 | 경로 | 주요 내용 |
| --- | --- | --- |
| 프로젝트 개요 | `AgentRule/Project/Onepiece/PROJECT_OVERVIEW.md` | 프로젝트 목표, 모듈 구성, 일정, 빌드 및 배포 환경 |
| 코딩 컨벤션 | `AgentRule/Project/Onepiece/CODING_CONVENTIONS.md` | Onepiece 고유 C++/언리얼 규칙, 로깅 시스템, 매크로 정책 |
| Doxygen 설정 | `AgentRule/Project/Onepiece/DOXYGEN_SETUP.md` | 문서화 파이프라인, GitHub Pages 배포, Doxygen 구성 항목 |

각 에이전트는 작업 시작 전에 위 문서를 확인하여 최신 규칙을 파악해야 하며, 변경 사항이 있을 경우 해당 문서를 우선적으로 갱신합니다.

## 적용 범위

- **대상 에이전트**: Gemini, Claude, 범용 에이전트(`AGENTS.md`).
- **적용 영역**: 언리얼 엔진 5.6 기반 C++ 코드, CoffeeLibrary/LatteLibrary 모듈, 문서 작성, Doxygen 배포.
- **업데이트 방식**: 프로젝트 정책이 변경되면 먼저 본 문서를 수정하고, 각 에이전트 가이드에서는 링크만 유지합니다.

## Doxygen 노출 정책

- 에이전트 가이드는 내부 협업용으로만 사용하며, Doxygen 공개 문서에는 포함하지 않습니다.
- Doxygen에서 에이전트 문서가 보인다면 `Doxyfile`의 `INPUT` 목록에 `AgentRule/` 또는 개별 가이드가 추가되었는지 확인하고 즉시 제거
  하세요.
- 신규 에이전트 문서를 작성할 때는 본 가이드와 개별 모델 가이드만 갱신하고, Doxygen 설정은 변경하지 않습니다.

## 사용 지침

1. 에이전트는 작업 전에 **프로젝트 개요**, **코딩 컨벤션**, **Doxygen 설정** 문서를 차례로 확인합니다.
2. 공통 규칙이 변경되면 본 문서와 참조 문서를 동시에 업데이트하고, 각 에이전트 전용 문서에서는 링크만 최신 상태로 유지합니다.
3. Doxygen 배포 결과에 에이전트 가이드가 누락되면 `Doxyfile` 입력 경로와 문서 링크가 올바른지 본 문서의 체크리스트로 검증합니다.

> **TIP**: 에이전트 전용 문서에는 공통 지침을 재서술하지 말고, 본 문서 링크와 모델 특화 지침만 남겨 두세요.
