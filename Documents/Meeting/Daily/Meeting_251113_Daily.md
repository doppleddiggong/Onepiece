## 2. 논의 요약

### 2.1. 주요 논의
- OT 기간 동안 신규 멤버는 실제 작업을 통해 역할과 강점을 파악하도록 하고, 야간·주말 작업은 화면 공유 등 투명성 확보를 전제로 운영하기로 함.
- 입국 절차/택시/호텔/식사/관광을 잇는 시나리오를 중심으로 읽기-쓰기-말하기 흐름을 짜고, 아스트로봇 계열 플레이어와 거대 NPC의 캐릭터 품을 정리함.
- 토이 프로젝트 Flow를 본 프로젝트 플로우로 전환해 실질적인 문서화와 시연 준비에 활용하고, 오늘 오후부터 본 프로젝트 Flow 작업 진행.

### 2.2. 기술 / 기획 관련 논의
- Base64 통일 데이터를 기준으로 이미지·음성(STT/TTS) 송수신을 설계하며, API 스펙을 문서화하기 위한 자료를 정리 중.
- GPT 외 한국어 특화 로컬 모델과 VectorDB 기반 RAG 구조를 병행 검토하여 법률/규정 등 변동성이 큰 데이터와 웹서치 기반 콘텐츠를 명확히 분리.
- 클라이언트는 시퀀스/씬 플로우와 커밋 자동화, AI 팀은 Whisper STT/TTS 테스트 및 API 문서화 초안을 준비하며 협업 선순환을 구축.

### 2.3. 일정 / 운영 관련 논의
- 내일 시나리오 기반 브리핑과 클라이언트·AI 합동 검토를 진행하고, 주말에는 PPT/Flow 다이어그램과 시나리오 비주얼 작업에 집중.
- 월요일 교수님 리뷰 및 기술 설계 1차 완성, 화요일 발표를 목표로 오전까지 초안 완성하여 발표 준비.
- 오늘 오후 DevOps 세팅(커밋 메시지 자동화, 지라/컨플루언스 연결)을 마무리하고, API/Flow 문서는 주말 동안 통합 정리.
