## 2. 논의 요약

### 2.1. 협업 툴 선정
- **Jira 도입 결정:** Atlassian Jira를 이용해 프로젝트 관리 체계화 예정.
- **초대 메일 정리:** 재진 님이 민우 님과 협업하여 Jira·Notion 초대 메일 정리.
- **목표:** DevOps형 협업 관리 환경 구축 (Jira + Notion 통합 운영).

### 2.2. 일정 공지
- **다음 주 화요일:** 기획 발표 후 팀 회식 (장소: 생활맥주).

### 2.3. 팀 운영 프로세스
- **아침 미팅 프로토콜:** 09:30 공지, 각자 데일리 스크럼 작성 필수, 업무 공유 및 전일 진행 사항 점검 목적.

### 2.4. 당일 주요 작업
- **기획 정리 및 초안 작성:** 파이널 프로젝트 기획서 초안 완성 목표, 메타버스·AI 융합 방향성 중심.

### 2.5. 비주얼 콘셉트
- **아스트로봇 스타일 비주얼:** Unreal Stack O Bot 리소스를 참고해 비주얼 프로토타입 구성, 방향: 캐주얼하면서 친근한 SF풍 메타버스 환경.

### 2.6. 핵심 프로젝트 아이디어
- **“메타버스 안에서 교육이 일어나는 시스템”** (AI 튜터 + 실시간 상호작용형 학습 환경)
- **기본 방향:** 오프라인 교육을 온라인·메타버스 형태로 전환, 코딩/AI 교육을 게임화·시각화, 초등학생용 AI 코딩 교육을 주요 타깃으로 설정.
- **핵심 구성:**
  1. 그래프 기반 코딩 환경 (스크래치 유사)
  2. n8n 기반 로직 시각화 (노드 연결형 교육)
  3. 챗봇 교육 모듈 (단계별 챗봇 생성 과정 학습, 에이전트 제작 실습)
  4. RAG 기반 데이터형 AI 교육 (로컬 데이터 학습 → RAG 설계 및 구축 실습)
  5. 비전공자 대상 파이썬 입문형 학습 경로 (자연어 기반 코드 생성 및 설명형 피드백)
  6. AI 설계형 에이전트 교육 (프롬프트·RAG·액션 연동형 Agent Builder 체험)
  7. 외국인 대상 한국어 튜터 (AI 대화형 튜터 시나리오 확장 (TTS/STT 연동))
- **확장 아이디어:** 코딩·요리·과학 등 다양한 주제의 “온라인 튜터 시리즈”, 아두이노 등 실제 하드웨어 연동형 교육 콘텐츠 가능성.

### 2.7. 콘텐츠 방향 요약
- **핵심 주제:** AI 기반 메타버스 교육 플랫폼
- **핵심 기능:** 챗봇/에이전트 제작 실습, RAG 기반 데이터 학습
- **대상:** 초등생~일반 사용자
- **기술 스택:** Unreal Engine + n8n + Python + RAG + STT/TTS
- **교육 목표:** 비전공자도 “AI를 설계하는 경험” 제공
- **비주얼 콘셉트:** 아스트로봇 스타일의 친근한 메타버스 공간
