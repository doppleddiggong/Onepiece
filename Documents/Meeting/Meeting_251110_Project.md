# 브레인스토밍 1차 회의록 요약 (2025-11-10)

## 🧭 회의 주제
노인 복지와 AI 기술을 결합한 서비스 방향 탐색 및  
멀티플레이 구조 분리, 실버 세대 데이터 활용 방안 논의

---

## 1️⃣ 주요 논점 정리

| 구분 | 내용 |
|------|------|
| **핵심 의도** | 실버 세대의 실제 복지 문제를 AI 기술로 해결하는 콘텐츠 기획. |
| **주요 쟁점** | ① 복지관 디바이스 재등록 등 접근성 문제<br>② AI·RPC를 함께 적용할 때의 성능 저하<br>③ 멀티플레이 구조와 일반 통신의 구분 필요 |
| **아이디어** | - 치매 예방 게임 형태의 복지용 앱<br>- 가정용 재활·운동 보조 시스템<br>- 손주 AI / 육아 시뮬레이터<br>- 동화책 제작 및 애니메이션 자동화<br>- 보이스피싱 탐지형 AI 교육 게임 |
| **기술 제안** | STT/TTS, MediaPipe 기반 모션 인식, RPC를 이용한 캐릭터 반응, AI 생성 이미지(Lora·SD) 활용 등 |
| **데이터 활용 방향** | 실버 세대의 생활·행동 데이터 기반 맞춤 복지 서비스 설계 |

---

## 2️⃣ 결정 및 후속 과제

| 항목 | 내용 |
|------|------|
| **결정 사항** | “AI 기반 노인 복지 앱”을 중심 콘셉트로 설정.<br>기술 중심보다 **접근성·지속성·실용성**을 우선 고려. |
| **후속 과제** | - 세부 기능 정의 (치매 예방/동화책/손주AI 등)<br>- RPC+AI 통합 구조 검토<br>- MediaPipe 적용 타당성 실험 |
| **기술 검토 항목** | STT→TTS 피드백 루프, RPC 네트워크 분리, 사용자 행동 데이터의 구조화 및 재학습 모델 |

---

## ✅ 요약 결론
팀은 노인 복지, 교육, 게임 요소를 결합한 **AI 기반 실버 케어 앱** 개발을 1차 목표로 확정함.  
기능 설계에서는 **MediaPipe + STT/TTS + RPC 구조**를 중심으로 접근하고,  
차기 회의에서 구체적 기능 세분화와 기술 검증을 진행하기로 함.
